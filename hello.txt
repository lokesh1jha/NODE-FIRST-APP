What are the core modules of node js?
    Core modules are simple or complex functionality organized in Node.js to be used when needed. Like a library function.
Explain the use of each one of them in detail.
    http --> creates an HTTP server in Node.js.
    assert --> set of assertion functions useful for testing    
    fs --> used to handle file system.  
    path --> includes methods to deal with file paths.
    process --> provides information and control about the current Node.js process.
    os --> provides information about the operating system.
    querystring --> utility used for parsing and formatting URL query strings.
    url --> module provides utilities for URL resolution and parsing.
What are relative path and absolute path? How to define them?
    Relative Path provides address to the file with reference to the current directory. Starts with "/"
    Absolute Path provides address to the file with complete address.
